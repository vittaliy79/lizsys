
# lizsys
Lizing system

# Project stack
JavaScript  
Database: SQLite

# Init the project
To init the project run the following

    ./init_project.cmd  


# Starting the project
To start the project run two commands inside two separate terminals

    ./start_backend.cmd 
and

    ./start_frontend.cmd  


Open the browser and go to the address:  
http://localhost:5173/
